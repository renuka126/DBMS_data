# SQL_codes

A collection of SQL queries, scripts, and PL/SQL examples written while learning database management systems.

## About this repo

There's always something that amazes you and increases your interest in learning — as part of my learning journey, I've been exploring various concepts related to data, and SQL has been one of the most fun parts of it!

## What's inside

| File | Topic |
|---|---|
| `create_table.sql` | DDL — table creation, constraints |
| `inserting_values.sql` | INSERT statement variations |
| `groupby_having.sql` | `GROUP BY`, `HAVING`, aggregate functions |
| `joins.sql` | INNER, LEFT, RIGHT, FULL joins |
| `subquery.sql` | Nested/subquery examples |
| `pl_sql.sql` | PL/SQL basics — procedural blocks, control flow |
| `functions.sql` | User-defined SQL functions |
| `cursor.sql` | Explicit cursor declaration and usage |
| `transaction.sql` | Transaction control basics, `COMMIT` |
| `rollback_transaction.sql` | `ROLLBACK` usage and scenarios |
| `savepoint.sql` | `SAVEPOINT` for partial rollbacks |
| `schedules.sql` | Scheduling recurring jobs (`DBMS_SCHEDULER` / MySQL `EVENT`) |
| `triggers.sql` | Trigger creation — enforcing rules, auditing |
| `views.sql` | Creating and using views |
